# Quiz App

A simple and interactive Quiz App built using HTML, CSS, and JavaScript.

## Features

- Multiple choice questions
- Instant answer checking
- Score calculation
- Next question button
- Play again option
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```bash
quiz-app/
│── index.html
│── style.css
│── script.js
```
